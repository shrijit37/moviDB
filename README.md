MERN stack based movie database where users can signup,comment,rate movies with admin panel.






## Run Locally

Clone the project

```bash
  git clone https://github.com/shrijit37/moviDB.git
```

Go to the project directory

```bash
  cd moviDB
```

Install dependencies

```bash
  npm install
```

### Start the server
#### To run frontend 

```javascript
npm run frontend
```

#### To run backend 

```javascript
npm run backend
```
#### To run full stack 

```javascript
npm run fullstack
```


## files not include

- .env
- /node_modules
